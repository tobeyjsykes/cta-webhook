Privacy Policy

Last updated: June 21, 2026

This policy describes how the automated tool connected to the Instagram account @tjs.training handles data. This tool is operated for a single Instagram professional account and is not a public-facing service.

What this tool does

When someone comments on a post from @tjs.training, this tool checks the comment text against a small list of keywords. If a keyword matches, it automatically sends that person a private reply (Direct Message) containing a link to a relevant research study.

Information we collect

To do this, the tool receives the following information from Instagram for each comment, via Instagram's official Webhooks and Messaging APIs:

  • The text content of the comment
  • The Instagram user ID and username of the person who commented
  • The ID of the post and comment

We do not collect any information beyond what Instagram provides through these standard API events, and we do not use tracking pixels, cookies, or any other form of tracking.

How we use this information

The comment text is used only to determine whether it matches a configured keyword. If it matches, the commenter's Instagram user ID is used solely to send them a single automated private reply with the relevant study link. The information is not used for advertising, profiling, or any purpose beyond this single automated reply.

Data storage and retention

We do not maintain a database of commenters or their messages. Comment events are processed in real time to decide whether to send a reply, and are not stored afterward beyond standard, short-lived operational logs kept for debugging purposes. These logs are not used for any other purpose and are periodically cleared.

A separate, internal list of keyword-to-study-link mappings is maintained to configure what the tool replies with. This list does not contain any personal data about commenters.

Data sharing

We do not sell, rent, or share data with third parties for marketing or advertising purposes. Data passes only through the infrastructure required to operate this tool (Meta's Instagram Platform APIs and the hosting/automation services used to run it).

Your rights

If you have commented on a post from @tjs.training and would like any associated data removed or have questions about this policy, contact us using the details below.

Changes to this policy

This policy may be updated occasionally to reflect changes in how the tool operates. The "last updated" date above will reflect the most recent revision.
